## 🚀 Featured Project — DocMind AI

<div align="center">

[![DocMind AI](https://img.shields.io/badge/🧠_DocMind_AI-RAG_Chatbot-7c3aed?style=for-the-badge)](https://github.com/Sakshi983-cmd/-RAG-CHATBOT)
[![Live Demo](https://img.shields.io/badge/🔴_LIVE-Streamlit_App-FF4B4B?style=for-the-badge)](https://nhvb3mmj3ngfnosel4f9hf.streamlit.app)

</div>

> An AI-powered chatbot that reads your documents and answers questions in **real-time** using RAG + streaming.

```
PDF/TXT → Chunking → Embeddings → FAISS → Retriever → Groq LLM → Streaming UI
```

| Component | Technology |
|---|---|
| 🤖 LLM | `llama-3.1-8b-instant` via Groq |
| 🔢 Embeddings | `all-MiniLM-L6-v2` |
| 🗃️ Vector DB | `FAISS` |
| 🖥️ UI | `Streamlit` with custom dark theme |
